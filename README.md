# gzh CLI  
脚手架工具开发学习
## 支持功能

* 快速创建项目模板

## Install  

```bash

# 全局安装cli
npm install gzh-cli -g

# 本地node版本如安装在系统目录下，可能需要sudo执行
sudo npm install gzh-cli -g

# 如果安装有`node-sass`报错 请尝试:
npm install gzh-cli -g --unsafe-perm  
```

## Usage  

```bash
gzh-cli -v # 查看版本
gzh-cli -h # 查看帮助

# 创建新项目
gzh-cli create wxapp-test
```
